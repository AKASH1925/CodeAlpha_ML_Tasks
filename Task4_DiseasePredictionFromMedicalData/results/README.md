# Results Directory
This directory contains the output charts generated by `predict.py`.
## Charts
| File | Description |
|------|-------------|
| `accuracy_comparison.png` | Bar chart comparing accuracy across all models and datasets |
| `precision_comparison.png` | Precision comparison |
| `recall_comparison.png` | Recall comparison |
| `f1_comparison.png` | F1 score comparison |
| `confusion_matrices.png` | Confusion matrices for all models on all datasets |
| `roc_curves.png` | ROC curves with AUC scores |
