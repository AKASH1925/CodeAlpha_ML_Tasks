# Task 1: Credit Scoring Model
## Objective
Predict an individual's **creditworthiness** using past financial data.
## Approach
- Use **classification algorithms** like Logistic Regression, Decision Trees, and Random Forest
- Feature engineering from financial history
- Model accuracy assessment using **Precision, Recall, F1-Score, ROC-AUC**
## Dataset
The model uses financial features including:
- Income, debts, payment history
- Credit utilization, account age
- Employment status, number of credit accounts
> **Note:** This project uses a synthetic dataset for demonstration. For production, use real datasets from [UCI ML Repository](https://archive.ics.uci.edu/ml/datasets/statlog+(german+credit+data)) or [Kaggle](https://www.kaggle.com/datasets/uciml/german-credit).
